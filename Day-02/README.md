# Operators  =>
    Operators are used to perform the operations between two operands .
1. Arithmetic Operators (+,-,*,/,%,**)
2. Assignment Operators (=,+=,-=,*=,/=,%=)
3. Relational Op / Comparision (<,>,<=,>=,==,===,!=)
4. Bitwise Op (&,|,! , <<,>>)
5. Logical Op (&& , || ,! )
6. Increment/ Decrement  (++i, i++,--i,i--)
7. Ternary Op / Conditional 
((condition)?statement1:statement2)
